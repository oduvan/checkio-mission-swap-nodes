Ваша задача поменять местами парные элементы списка ({% if interpreter.slug == "js-node" %}Array{% else %}Iterable{% endif %}). Если вам дам список из 4х элементов, то ваша функция должна вернуть такой же список, только в нем поменяны местами первый со вторым и третий с четвертым элементы.

Если элементов не парное количество, то последний, непарный, элемент оставить на своем месте. Пустой список должен остаться пустым.

Input: {% if interpreter.slug == "js-node" %}Array{% else %}Iterable{% endif %}.

Output: {% if interpreter.slug == "js-node" %}Array{% else %}Iterable{% endif %}.

Example:

{% if interpreter.slug == "js-node" %} 
swapNodes([1, 2, 3, 4]) == [2, 1, 4, 3]
swapNodes([5, 5, 5, 5]) == [5, 5, 5, 5]
{% else %} 
swap_nodes([1, 2, 3, 4]) == [2, 1, 4, 3]
swap_nodes([5, 5, 5, 5]) == [5, 5, 5, 5]
{% endif %}
